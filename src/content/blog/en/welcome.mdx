# First Blog Post

Welcome! This is the first post of the static blog built with Astro.

## Features

- Markdown and MDX support
- Multi-language support (Korean, English, Japanese)
- GitHub Pages deployment
- Dark/Light theme

## Tech Stack

- Astro
- Tailwind CSS
- GitHub API

This blog stores posts as MDX files in a GitHub repository and can be written and edited from the web editor.
