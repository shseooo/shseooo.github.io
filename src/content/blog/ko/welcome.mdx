# 첫 번째 블로그 포스트

환영합니다! 이것은 Astro로 만든 정적 블로그의 첫 번째 포스트입니다.

## 특징

- Markdown 및 MDX 지원
- 다국어 지원 (한국어, 영어, 일본어)
- GitHub Pages 배포
- 다크/라이트 테마

## 기술 스택

- Astro
- Tailwind CSS
- GitHub API

이 블로그는 GitHub 리포지토리에 MDX 파일로 저장되며, 웹 에디터에서 작성하고 수정할 수 있습니다.
