설치 없이 브라우저에서 바로 돌아가는 도구 모음입니다. 입력한 데이터는 서버로 전송되지 않고 기기 안에서만 처리됩니다.

인코딩 / 디코딩

  • Base64 인코더/디코더 UTF-8·URL-safe 지원 Base64 변환
  • URL 인코더/디코더 퍼센트 인코딩 (component / 전체 URI)
  • HTML 엔티티 변환 특수문자 ↔ HTML 엔티티
  • Hash 생성기 MD5 · SHA-1/256/384/512
  • UUID 생성기 난수 기반 UUID v4 일괄 생성

텍스트 / 데이터

  • JSON 포맷터/검증기 정렬·압축·문법 오류 위치 표시
  • 정규식 테스터 실시간 매치 하이라이트·캡처 그룹
  • JWT 디코더 헤더/페이로드 디코딩 · HS256 검증
  • Unix Timestamp 변환 타임스탬프 ↔ 날짜 양방향
  • 컬러 변환기 HEX ↔ RGB ↔ HSL + 미리보기
  • 글자수/단어수 카운터 문자·단어·읽기 시간 통계
  • Diff 비교기 두 텍스트 줄 단위 차이 비교
  • 정규식 패턴 라이브러리 자주 쓰는 패턴 모음 + 즉시 테스트
  • 탭 → 줄바꿈 변환 구분자 ↔ 줄바꿈 자유 변환
  • 열 단위 중복 제거 특정 컬럼 기준 중복 행 제거
  • 텍스트 수 카운트 텍스트 길이(글자 수) 표시

  • 환율 변환기 실시간 환율 (ECB/Frankfurter)
